# **App Name**: MathQuest

## Core Features:

- Student Login: Secure login system for students to access lessons, quizzes, and track progress.
- Teacher Login: Secure login system for teachers to manage sessions, upload resources, and share tutoring links.
- Student Dashboard: Tracks completed lessons, quiz scores, and badges with motivational messages.
- Resources Page: Animated lessons, videos, interactive games, and downloadable worksheets organized by grade and topic.
- Tutoring Schedule: Calendar view of available teacher times with a booking form including student details.
- Teacher Dashboard: Allows teachers to manage student bookings, upload resources, and share tutoring links. It also includes session attendance tracking.
- Personalized Learning Path Generator: AI tool to analyze student performance and generate a tailored learning path of recommended lessons and quizzes to address weaknesses.

## Style Guidelines:

- Primary color: Cheerful blue (#42A5F5) to promote a sense of trust and intelligence.
- Background color: Very light blue (#E3F2FD), providing a calm backdrop that makes other elements pop.
- Accent color: Energetic orange (#FFB74D) for CTAs and interactive elements, drawing attention to key actions.
- Body and headline font: 'PT Sans' for a modern look with some warmth, which is suitable for both headlines and body text.
- Simple, friendly math-related icons with rounded edges to complement the site's playful design.
- Big, easily-tappable buttons with rounded shapes for intuitive navigation, ensuring a smooth experience on both desktop and mobile.
- Subtle, playful animations upon completing lessons, quizzes, and earning badges to reinforce positive learning behaviors.
